# Stakeholder Information Requirements (Power BI)

The stakeholder requires a centralized overview to manage marketing expenditure efficiency and optimize user growth. The following information needs are addressed:

* **Core KPI Overview:** Insights into total user counts, actual bookings, and the resulting conversion rate (CR).
* **Channel Performance:** Identification of top-performing marketing providers (e.g., Google, Direct, Facebook) by correlating volume (Total Users) and quality (Conversion Rate) via bubble chart analysis.
* **Platform Optimization:** Analysis of user entry points by device (Web, iOS, Android) and conversion performance to prioritize technical optimizations.
* **Geographic & Seasonal Trends:** Monitoring booking volumes by destination country over time (monthly).
* **Target Group Segmentation:** Understanding user demographics (age, gender, language) to tailor marketing messaging.
* **Dynamic Filtering:** Capabilities to slice data by time periods, demographics, and device types to perform root-cause analysis on fluctuations.
